# Advent of Code 2025 Solutions

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Rust.

## About

Advent of Code is an annual set of Christmas-themed programming puzzles that can be solved in any programming language. This repository contains my attempts at solving the 2025 edition.
